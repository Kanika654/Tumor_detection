# Tumor Detection using Deep Learning and cGAN

##  Overview
This project focuses on tumor detection using deep learning and Conditional GANs (cGAN). The model generates synthetic medical images to handle class imbalance and improves classification accuracy using transfer learning with ResNet-18.

---

##  Features
- Tumor image classification
- cGAN-based data augmentation
- ResNet-18 transfer learning
- Handles imbalanced datasets
- PyTorch implementation

---

## Technologies Used
- Python
- PyTorch
- OpenCV
- NumPy
- torchvision
